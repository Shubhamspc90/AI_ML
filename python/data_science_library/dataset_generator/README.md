# Dataset Generator

## 📌 Purpose

This folder contains scripts and notebooks to generate realistic datasets for Data Science and Machine Learning projects.

Instead of using small dummy datasets, we generate large, industry-style datasets with realistic values.

---

## 📂 Datasets Generated

- sales.csv
- students.csv
- covid.csv
- housing.csv
- employees.csv

Each dataset can also be exported as an Excel (.xlsx) file.

---

## 📊 Features

- Large datasets (5,000–100,000+ rows)
- Realistic Indian names
- Indian cities and states
- Multiple product categories
- Random but meaningful values
- Missing values (for data cleaning practice)
- Duplicate records
- Outliers
- Date and time information
- Business-ready structure

---

## 📁 Output Structure

datasets/
├── raw/
├── processed/
└── reports/

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Faker
- OpenPyXL

---

## 🚀 How to Use

1. Open the notebook.
2. Run all cells.
3. Generated datasets will be saved inside the `datasets/` folder.

---

## 🎯 Goal

Generate realistic datasets that can be used for:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Machine Learning
- Portfolio Projects
- Interview Practice
